This project consists of several parts. Each part will be discussed into more detail below.

/coco: 		Python scripts to process the MSCOCO dataset.
/documents:	Source files for the thesis.
/downloads:	Directory to which the MSCOCO dataset is downloaded as well as other files that are used.
/input:		Directory that contains files that could be used as an input for parts of this project.
/legacy:	[DEPRECATED] Old code for a failed model.
/models:	Models of deep neural networks.
/networks:	Various networks to perform various tasks.
/notebooks:	IPython notebooks containing code for fiddling with various parts of the project.
/output:	Directory that could contains output files.
/scripts: 	Shell scripts to run parts of the experiment. More info inside.
/server:	Light-weight server to interface with the parts of the project.
